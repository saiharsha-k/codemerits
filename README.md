## Local Development

1. Clone the repository:
```bash
git clone https://github.com/saiharsha-k/codemerits.git
cd codemerits
```

2. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
# Visit http://localhost:8000
```
