Contribute to CodeMerits

Help us build the best curated resource platform for developers. We welcome contributions from everyone!

Ways to Contribute

  • Submit Resources: Share valuable coding resources, tutorials, or articles
  • Report Issues: Found a broken link or outdated resource? Let us know
  • Improve Documentation: Help us make guides clearer and more helpful
  • Suggest Features: Have an idea to improve the platform? We'd love to hear it

How to Submit

1. Fork the repository on GitHub

2. Add your resource in the appropriate category folder

3. Follow our resource format (title, description, link, tags)

4. Submit a Pull Request with a clear description

5. Wait for review - we typically respond within 48 hours

Contribution Guidelines

  • Resources must be free or have a quality free tier
  • Content should be up-to-date and actively maintained
  • Include a brief description explaining why the resource is valuable
  • Check for duplicates before submitting
  • Use clear, descriptive commit messages
